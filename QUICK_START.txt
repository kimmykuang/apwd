🔴 优先级 P0: Xcode（必需，强烈推荐）
────────────────────────────────────────
   为什么需要：启用完全自闭环自动化测试

   安装方法：
   1. 打开 App Store
   2. 搜索 "Xcode"
   3. 点击下载（约 15GB，1-2小时）

   安装后配置：
   sudo xcodebuild -license accept
   sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer

   然后打开 Xcode -> Settings -> Platforms -> 下载 iOS


🟡 优先级 P1: Android Studio（可选，跨平台测试）
────────────────────────────────────────
   为什么需要：测试 Android 兼容性

   安装方法：
   brew install --cask android-studio

   然后启动并选择 "Standard" 安装

📚 使用流程
═══════════════════════════════════════════════════════════════════

1️⃣ 验证环境
   ./verify_setup.sh

2️⃣ 运行自动化测试
   ./autonomous_test_runner.py ios        # iOS 测试
   ./autonomous_test_runner.py android    # Android 测试
   ./autonomous_test_runner.py ios android  # 全平台测试

3️⃣ 查看测试结果
   cat test_results/*_report.md           # 测试报告
   open test_results/*_screenshots/       # 截图
   cat test_results/*_analysis.json       # AI 可读的分析结果
